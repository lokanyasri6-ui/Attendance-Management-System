# 📚 Attendance Management System

## 📝 Project Description

The Attendance Management System is a simple Python application that helps manage student attendance records. Users can add students, mark attendance, and view attendance details through a menu-driven interface.

---

## ✨ Features

✅ Add Student Names  
✅ Mark Attendance (Present/Absent)  
✅ View Attendance Records  
✅ Simple Menu-Driven Interface  
✅ Beginner-Friendly Python Project

---

## 🛠️ Technologies Used

- 🐍 Python 3

---

## 📖 Concepts Covered

- 📋 Lists
- 📂 Dictionaries
- 🔄 Loops
- 🔀 Conditional Statements
- ⌨️ User Input and Output

---


## 🚀 How to Run

--
### 1️⃣ Install Python
Make sure Python 3 is installed on your system.

### 2️⃣ Save the Program
Save the code as:

```bash
attendance.py
```

### 3️⃣ Open Terminal/Command Prompt

Navigate to the project folder.

### 4️⃣ Run the Program

---

## 🎯 Menu Options

### ➕ Add Student
- Enter the student's name.
- Student is added to the system.

### ✔️ Mark Attendance
- P = Present
- A = Absent

### 📊 View Attendance
- Displays all attendance records.

### ❌ Exit
- Closes the application.
